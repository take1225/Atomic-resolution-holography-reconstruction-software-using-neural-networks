Software Summary
Atomic image reconstruction software for atomic resolution holography using neural networks.

Software Operating Environment
The software uses 3D-AIR-IMAGE, created by Dr. Matsushita.
In addition, api_3d_air_image.py is required to access each 3D-AIR-IMAGE data from the Python side.
api_3d_air_image.py exists in the 3D-AIR-IMAGE download file.

※It is still incomplete.

ソフトウェアの概要
ニューラルネットワークを利用した原子分解能ホログラフィーの原子像再構成ソフトウェアです。

ソフトウェアの動作環境
このソフトウェアの使用には松下博士の3D-AIR-IMAGEを使用します。
また、Python側から3D-AIR-IMAGEの各データにアクセスするためにapi_3d_air_image.pyが必要です。
api_3d_air_image.pyは3D-AIR-IMAGEのダウンロードファイルに存在しています。
※まだ未完成です。
